Detailed Description

Return a value that describes the local lid driven cavity problem domain stored in a process, for testing purposes

Returns
The value that describes an aspect of the local lid driven cavity problem domain
Note
Returned values describe the local domain of a process, which are not necessarily the same as the global domain (unless MPI ranks = 1)
