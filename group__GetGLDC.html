Detailed Description

Return a value that describes the global lid driven cavity problem domain, for testing purposes

Returns
The value that describes an aspect of the lid driven cavity problem domain
Note
The returned global values will be same across all processes. The member functions without Global in the name are parameters that are the same regardless of the problem discretisation.
